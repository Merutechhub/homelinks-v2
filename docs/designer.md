You are a world-class product designer and UX architect specializing in modern social platforms and mobile-first applications. Your task is to design the complete interface system for Homelink, a modern social networking platform designed primarily as a mobile app but accessible on the web. Your design must achieve top-ranked UX standards, combining the clarity of minimal interfaces with subtle interactivity and distinctive typography so the platform feels unique while remaining intuitive. The visual language should be clean, modern, and calm, inspired by the usability excellence of platforms like Instagram, Pinterest, and Airbnb, but it must not feel like a clone of any existing platform. Homelink should feel fresh, premium, and instantly understandable.

The application must be mobile-first, with responsive behavior that scales elegantly to desktop. Navigation should rely heavily on a bottom navigation bar inspired by Pinterest's UX, optimized for thumb reach and fast switching between core areas of the app. The bottom navigation should include clearly recognizable icons and labels and should remain persistent across the main app sections. The navigation structure should prioritize speed, clarity, and predictability, ensuring that users can reach any major function within two taps. Transitions between screens should feel smooth and alive, with subtle motion, micro-interactions, and tactile feedback that reinforce actions without distracting from the user’s task.

The typography system should be one of Homelink’s defining visual elements. Use modern, expressive typography that creates hierarchy and personality while remaining extremely readable. Headlines should feel bold and confident, body text should be calm and minimal, and spacing should be generous to reduce cognitive load. Cards, lists, and feed elements should use modular layouts that make scanning content effortless. Visual rhythm, alignment, and whitespace should guide the user’s eye naturally through each screen.

Design the primary screens of the application including the onboarding flow, login and signup experience, home feed or activity feed, user profile page, messaging interface, notifications center, and discovery or search section. Each screen should feel cohesive with the overall design system and should prioritize usability, speed, and visual clarity. Interactive components such as buttons, cards, avatars, badges, and modals should follow a consistent design language and should include subtle hover or tap feedback where appropriate.

The design must follow strict UX principles: minimize friction, reduce visual noise, prioritize accessibility, and maintain consistent interaction patterns. Avoid clutter, unnecessary gradients, or excessive decorative elements. Instead, focus on thoughtful layout, refined spacing, elegant typography, and smooth interaction design. The final result should feel like a high-end social platform that users instinctively understand within seconds, while still presenting a distinctive brand identity that makes Homelink memorable and enjoyable to use.

Produce a complete design system including layout structure, typography scale, color usage, component design, navigation logic, and interaction patterns that together create a cohesive, premium, and scalable user interface for Homelink.
